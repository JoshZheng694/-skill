# 通用教材复习资料生成器

从原始教材PDF到最终背诵资料的端到端全自动流水线。

## 完整闭环

```
📖 原始教材PDF (可能几百页)
      │
      ▼
┌─────────────────────────────────────┐
│ 阶段〇：PDF切分 + 批量OCR             │
│ • 自动将PDF切成小块（每块~10页）       │
│ • 并行OCR转MD（3个worker，自动检测扫描型）│
│ • 输出：42个 chunk_NN_pXXX-pYYY.md   │
└─────────────────────────────────────┘
      │
      ▼
┌─────────────────────────────────────┐
│ 阶段一：10代理并行提取                │
│ • 每个代理分配4-5个chunk + 完整提纲模板 │
│ • 逐考点在MD中搜索教材原文，填入模板    │
│ • 结果立即存本地，不返回主代理          │
└─────────────────────────────────────┘
      │
      ▼
┌─────────────────────────────────────┐
│ 阶段二：10代理并行审阅                │
│ • 逐考点核对教材原文准确性             │
│ • 替换所有"……"/"..."为完整原文        │
│ • 补充遗漏考点，修正页码偏移           │
│ • 区分"教材原文" vs "课堂补充"         │
└─────────────────────────────────────┘
      │
      ▼
┌─────────────────────────────────────┐
│ 阶段三：2代理元审阅                   │
│ • 交叉检查全部10个审阅输出             │
│ • 全局一致性验证 + 考前Checklist       │
│ • 产出问题清单 + 修复建议              │
└─────────────────────────────────────┘
      │
      ▼
┌─────────────────────────────────────┐
│ 阶段四：1代理整合交付                 │
│ • 合并所有审阅修正到最终文档           │
│ • 统一页码/格式/标注                  │
│ • 产出：最终背诵资料 + CHANGELOG       │
└─────────────────────────────────────┘
```

## 阶段〇：PDF切分 + 批量OCR

### PDF切分

使用PyPDF2将大PDF切成小块（每块10页）：

```bash
python -c "
from PyPDF2 import PdfReader, PdfWriter
from pathlib import Path

pdf = PdfReader('教材.pdf')
chunk_size = 10
for start in range(0, len(pdf.pages), chunk_size):
    end = min(start + chunk_size, len(pdf.pages))
    writer = PdfWriter()
    for i in range(start, end):
        writer.add_page(pdf.pages[i])
    out = f'原始文件/chunk_{start//chunk_size+1:02d}_p{start+1}-p{end}.pdf'
    Path(out).parent.mkdir(exist_ok=True)
    with open(out, 'wb') as f:
        writer.write(f)
    print(f'{out} ({end-start}页)')
"
```

### 批量OCR转Markdown

对每个PDF分块调用 `pdf-to-markdown` 技能脚本，3个worker并行：

```bash
# 单个分块转换
python .claude/skills/pdf-to-markdown/scripts/convert_pdf.py \
  原始文件/chunk_01_p1-p10.pdf \
  原始文件/chunk_01_p1-p10.md \
  --dpi 200

# 批量并行（3 worker，禁止超过3个）
# 使用 _run_3p.py 或手动分批
```

关键约束：
- DPI 200（平衡速度与精度）
- 最多3个OCR worker并行（32GB RAM限制）
- EasyOCR中文+英文模型，CPU模式
- 自动跳过已有MD的分块（去重）
- RAM > 90% 杀进程，新任务前清残留Python进程

### OCR分块命名规范

```
原始文件/chunk_01_p1-p10.md
原始文件/chunk_02_p11-p20.md
...
原始文件/chunk_42_p411-p419.md
```

## 阶段一：10代理提取

主代理执行 `/do-agent` 模式编排。

### 工作目录

```
agent_tasks/{task_name}_YYYYMMDDHH/
├── plan.md              ← 执行计划
├── 提纲模板.md           ← 复习指南副本（供子代理填充）
├── partial_A{N}.md      ← 10个子代理输出
└── 经济史完整复习提纲.md  ← 最终输出
```

### 分块分配

| Agent | 分块 | 页码范围 |
|-------|------|----------|
| A1 | chunk_01~04 | p1-40 |
| A2 | chunk_05~08 | p41-80 |
| A3 | chunk_09~12 | p81-120 |
| A4 | chunk_13~16 | p121-160 |
| A5 | chunk_17~20 | p161-200 |
| A6 | chunk_21~24 | p201-240 |
| A7 | chunk_25~28 | p241-280 |
| A8 | chunk_29~32 | p281-320 |
| A9 | chunk_33~36 | p321-360 |
| A10 | chunk_37~42 | p361-419 |

### 提取代理规范

每个代理：
1. 自己读取分配的chunk + 完整复习指南模板
2. 逐考点搜索教材原文，填入模板
3. 格式：`**教材原文**（第X页）："完整原文"`
4. 未找到标注"（未覆盖）"
5. **立即保存结果到本地，不返回主代理**
6. 返回状态摘要：找到X个考点、未覆盖Y个

## 阶段二：10代理审阅

### 审阅分工

| 代理 | 板块 | 教材分块 | 重点 |
|------|------|---------|------|
| R01 | 板块一前半 | chunk_01~04 | 修复省略号，修正页码偏移 |
| R02 | 板块一后半+课堂补充 | chunk_03~08 | 填补遗漏考点 |
| R03 | 板块二前半 | chunk_07~12 | 验证赋税制度演变链 |
| R04 | 板块二后半 | chunk_05~12 | 找遗漏考点教材出处 |
| R05 | 板块三 | chunk_07~14 | 验证商业/货币/丝绸之路 |
| R06 | 板块四前半 | chunk_13~18 | 填补真空区+验证因果链 |
| R07 | 板块四后半+五开头 | chunk_17~24 | 近代→当代过渡 |
| R08 | 板块五 | chunk_21~32 | 补全缺失的教材原文 |
| R09 | 板块六+七 | chunk_31~42+导论 | 搜索理论来源+新发展 |
| R10 | 板块八+全局扫描 | chunk_37~42+全范围 | 前沿议题+省略号全盘 |

### 审阅质量标准

| 规则 | 说明 |
|------|------|
| 🔴 省略号零容忍 | 所有"……"/"..."必须替换为完整教材原文 |
| 🔴 页码统一 | 以教材印刷页码为准，消除PDF页码偏移 |
| 🟡 来源区分 | 教材无涉及标 `> ⚠️ 课堂补充：`，不得伪装为教材原文 |
| 🟡 逐条可查 | 每条"教材原文"引用必须在对应OCR分块中找到原文 |
| 🟢 相邻重叠 | 边界考点由两个代理覆盖，防止遗漏 |

## 阶段三：2代理元审阅

| 代理 | 范围 | 任务 |
|------|------|------|
| R-A | 板块一~四 | 覆盖完整性、审阅间矛盾、省略号验证、缺口状态 |
| R-B | 板块五~八+Checklist | R08深度验证（抽查原文）、R09理论搜索验证、全局一致性 |

## 阶段四：整合交付

1个教师代理：
- 读取全部10审阅 + 2元审阅报告
- 合并修正到最终文档
- 统一格式、页码、标注
- 产出：`经济史完整复习提纲_修订版.md` + `CHANGELOG.md`

## 输出格式

每个考点：
```
### N. 考点名称 【优先级：★必考/重点/了解】

**教材原文**（第X页）："完整原文引用，无省略号"

**考试要点**：……

**答题关键词**：……
```

课堂补充：
```
> ⚠️ 课堂补充：教材未设专门章节，以下为教师课堂讲授内容
```

## 内存约束

- 总RAM 32GB，3个EasyOCR Worker约15GB
- 禁止超过3个OCR Worker同时运行
- RAM > 90% 必须杀进程
- 新OCR任务前先清残留进程：
  ```powershell
  powershell -Command "Get-Process python -ErrorAction SilentlyContinue | Stop-Process -Force"
  ```
