## 审阅代理工作规范

### 每个审阅代理必须执行：

1. **读取提纲原文**：从 `经济史完整复习提纲.md` 中定位自己负责的板块，精读现有内容
2. **读取教材分块**：逐块读取自己分配到的教材OCR分块
3. **逐考点核对**：
   - 检查提纲中每个考点的"教材原文"引用是否**确实来自教材对应页码**
   - 检查引用中是否有 **"……" 或 "..."** 省略号——如果有，**必须找到教材原文完整补充**
   - 检查页码标注是否准确（统一使用教材印刷页码）
4. **补充缺失内容**：
   - 对于前一版本审阅报告中标注的缺失考点，在教材分块中搜索相关原文
   - 如果教材某章节确实讨论了某考点但之前未被提取，补充完整原文
   - 如果教材确实没有该考点（如纯课堂补充的理论），明确标注"教材无直接论述，以下为课堂讲授内容"
5. **输出格式要求**：
   - 每个考点输出结构：
     ```
     ### N. 考点名称 【优先级：★必考/重点/了解】
     
     **教材原文**（第X页）："完整原文引用，不使用省略号"
     
     **考试要点**：...
     
     **答题关键词**：...
     ```
   - 如果教材原文超过200字，选取最核心的完整段落，分多条"教材原文"标记
6. **立即保存**：将审阅完成的板块内容保存到 `agent_tasks/经济史复习提纲审阅补充_2026062117/review_R{N}.md`
7. **返回状态摘要**：只返回"审阅X个考点，发现Y处省略号已修复，补充Z处缺失内容，N个考点教材无直接论述"

### 省略号修复标准

- **严禁**出现：`"……"`、`"..."`、`"等等。"` 结尾的教材引文
- **必须**：完整摘录教材句子，或使用 `[中间N段略]` 明确标记跳过部分
- 对于实在过长的段落（>300字），使用两条或多条完整引文，中间加 `[……以上为核心段落，详细展开见教材第X-Y页]`

### 页码规范

- 统一使用**教材印刷页码**（即教材页脚标注的页码）
- 如果OCR分块中的页码与教材印刷页码存在偏移，以教材印刷页码为准
- 无法确定教材页码时，标注分块文件名（如 `chunk_15_p141-p150` 中的相对位置）
