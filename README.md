# study-guide-generator

从原始教材PDF → 最终背诵资料，全自动。

## 一句话

扔一本教材PDF进来，给你一份可以直接打印背诵的复习提纲。

## 完整流程

```
📖 原始PDF
  │  自动切分（每块~10页）+ 批量OCR转MD（扫描版也能识别）
  ▼
📄 42个Markdown分块
  │  10个AI代理并行，逐考点搜索原文填入提纲模板
  ▼
📝 初版复习提纲
  │  10个AI代理审阅（修复省略号、补遗漏、验页码、区分课堂补充）
  ▼
✅ 审阅修正版
  │  2个AI代理交叉检查（全局一致性、抽查原文准确度）
  ▼
⭐ 最终背诵资料 + 完整变更日志
```

## 怎么用

```bash
/study-guide-generator 把教材内容填入复习提纲，审阅补充完整
```

触发词：`整理复习资料` `做背诵资料` `考试复习` `划重点` `一键生成复习资料`

## 你需要准备

| 材料 | 格式 | 说明 |
|------|------|------|
| 教材 | PDF | 扫描版也行，自动OCR |
| 复习指南 | DOCX / MD | 考点列表 + 页码索引 |
| PPT（可选） | PDF / 图片 | 老师课堂补充 |

## 质量保证

- 🔴 省略号零容忍 — 所有引文替换为完整原文
- 🔴 页码精确 — 统一教材印刷页码
- 🟡 来源区分 — 教材原文 vs 课堂补充，⚠️ 标记分明
- 🟡 逐条可查 — 每条引文能在源文件中找到
